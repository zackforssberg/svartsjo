# Svartsjo

My first ever school project.

## Overview

Svartsjo is an HTML and CSS-based web project featuring multiple pages and galleries. This is my first "real" project ever, showcasing web design and styling skills.

## Project Structure

The project includes several pages with corresponding stylesheets:

- **index.html** / **index.css** - Main landing page
- **galleri.html** / **galleri.css** - Image gallery page
- **gården.html** / **gården.css** - "The Farm" page (Swedish content)
- **main.css** - Global styling

### Assets

- **bilder/** - Directory for storing images used throughout the project

## Technologies

- **HTML5** - Page structure and content
- **CSS3** - Styling and layout

## Getting Started

To view the project, simply open `index.html` in your web browser to navigate through the different pages.
